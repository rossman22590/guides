Adding Bootstrap to your site is a simple matter of importing the Bootstrap script and stylesheet in your pages' `<head>`. This is done in `motif.json`, located at the root of your project. Find the `head` section, and add the Bootstrap script and stylesheet tags, as indicated on [the official website](https://getbootstrap.com/):

```json
{
  "head": [
    "<link href='https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css' rel='stylesheet' integrity='sha384-GLhlTQ8iRABdZLl6O3oVMWSktQOp6b7In1Zl3/Jr59b6EGGoI1aFkw7cmDA6j6gD' crossorigin='anonymous'>",
    "<script src='https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/js/bootstrap.bundle.min.js' integrity='sha384-w76AqPfDkMBDXo30jS1Sgez6pr3x5MlQ1ZAGC+nuZB+EYdgRZgiwxhTBTkF7CXvN' crossorigin='anonymous'></script>"
  ]
}
```

You can now use Bootstrap throughout your JSX components:

```jsx
<div class="alert alert-danger" role="alert">
  A simple danger alert—check it out!
</div>
```
